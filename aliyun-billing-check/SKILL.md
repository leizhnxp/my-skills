# 阿里云消费查询技能

本技能帮助用户查询阿里云账单数据，展示预付费、后付费消费汇总，以及每月消费最高的3个实例和3个品类。

## 前置条件

- 已安装阿里云 CLI (`aliyun` 命令)
- 已配置阿里云访问凭证（AccessKey ID 和 AccessKey Secret 或 OAuth）

## 使用流程

### 1. 检查配置

首先检查阿里云 CLI 是否已配置：

```bash
aliyun configure list
```

如果未配置，提示用户先运行 `aliyun configure` 进行配置。

如果有多个 profile，询问用户使用哪个 profile。

### 2. 询问查询方式

**先询问用户想如何查询：**

```
请选择查询方式：
1. 查询最近N个月
2. 查询指定月份
```

根据用户选择：

- **如果选择 1（最近N个月）**：询问 "请输入需要查询的月数（不含当月，默认14）："
  - 用户输入数字则使用该数字
  - 用户直接回车则使用默认值 14

- **如果选择 2（指定月份）**：询问 "请输入月份（格式：YYYY-MM）："
  - 用户输入如 "2026-02" 这样的格式

### 3. 询问是否输出到文件

询问用户："是否需要将结果保存到文件？(y/n，默认n)"

- 如果用户选择 y，询问文件名（默认：aliyun_billing_YYYYMMDD_HHMMSS.md）

### 4. 执行查询

根据用户的选择调用脚本：

**查询最近N个月：**
```bash
python3 ~/.claude/skills/aliyun-billing-check/scripts/get_billing.py --profile <profile名称> --months <N> [--output <文件名>]
```

**查询指定月份：**
```bash
python3 ~/.claude/skills/aliyun-billing-check/scripts/get_billing.py --profile <profile名称> --month <YYYY-MM> [--output <文件名>]
```

#### 主要 API 调用

**查询月账单汇总：**
```bash
aliyun bssopenapi QueryBillOverview --BillingCycle YYYY-MM
```

**查询实例账单明细（用于获取实例ID和TOP3）：**
```bash
aliyun bssopenapi DescribeInstanceBill --BillingCycle YYYY-MM --MaxResults 300
```

### 5. 数据处理逻辑

脚本会自动处理：
1. 对每个月调用账单 API：
   - QueryBillOverview 获取汇总数据（预付费、后付费）
   - DescribeInstanceBill 获取实例级明细
2. 从返回结果中提取：
   - 总消费金额
   - 预付费金额
   - 后付费金额
   - 按实例聚合的消费明细
   - 按品类聚合的消费明细
3. 找出每个月消费最高的3个实例和3个品类
4. 汇总数据并生成表格
5. 同时输出到控制台和指定文件（如果有）

### 6. 输出格式

生成包含实例TOP3和品类TOP3的 Markdown 表格：

| 月份 | 总计(元) | 预付费(元) | 后付费(元) | 实例TOP1 | 实例TOP2 | 实例TOP3 | 品类TOP1 | 品类TOP2 | 品类TOP3 |
|------|----------|------------|------------|----------|----------|----------|----------|----------|----------|
| 2026-02 | 95543.46 | 78751.00 | 16792.46 | 云服务器 ECS ... i-xxx(32550.00) | 数字证书...(10438.40) | 云安全中心...(4944.87) | 云服务器 ECS(44390.43) | 数字证书...(10438.40) | 云数据库 RDS(8849.54) |
| ... | ... | ... | ... | ... | ... | ... | ... | ... | ... |
| **合计** | **XXXX.XX** | **XXXX.XX** | **XXXX.XX** | - | - | - | - | - | - |

TOP列格式为：`名称(金额)`

## 脚本参数说明

| 参数 | 说明 |
|------|------|
| `--profile` | 阿里云 CLI 配置的 profile 名称 |
| `--months` | 查询最近 N 个月的数据（不含当月） |
| `--month` | 查询指定月份（格式：YYYY-MM） |
| `--max-pages` | 最大分页数（安全上限，默认100页约30000条） |
| `--output`, `-o` | 输出结果到指定文件 |

## 示例

### 输入
"帮我看看阿里云最近的消费情况"

### 交互流程
1. 检查配置，发现有 OAuthProfile
2. 询问用户查询方式
3. 用户选择查询最近3个月
4. 询问是否保存到文件，用户选择保存
5. 执行脚本并输出表格
6. 同时保存到文件

## 注意事项

- 确保阿里云 CLI 有足够的权限访问 Billing API
- 账单数据可能有延迟，最近几天的数据可能不完整
- 如果遇到权限问题，提示用户检查 RAM 权限配置
- 查询大量月份数据可能需要较长时间，建议先查询1-3个月测试
